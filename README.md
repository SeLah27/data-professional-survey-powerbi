# PowerBi-Data-Survey-Analysis
This project analyzes a survey of data professionals to uncover insights about salaries, job roles, demographics, and industry challenges.
The dashboard provides an interactive overview of how different factors such as job title, country, and experience impact outcomes in the data field.

🛠️ Tools Used
Power BI
Data visualization & dashboard design.

📂 Dataset
Survey dataset of data professionals
Includes job roles, salary, gender, country, and job satisfaction metrics and many more.

🔍 Key Analysis
Average salary by job title
Distribution of survey participants by country
Gender distribution across roles
Favorite programming languages
Difficulty of entering the data field
Demographic insights (age, experience)

📈 Key Insights
Data Scientists tend to have the highest average salaries
Python is the most popular programming language among respondents
A large portion of participants report moderate difficulty entering the data field
Gender distribution varies significantly across job roles
The majority of respondents are concentrated in a few key countries

Designing interactive dashboards using Power BI
Transforming survey data into meaningful insights
Structuring data storytelling for business understanding
